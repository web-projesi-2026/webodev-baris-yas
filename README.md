# 13nisan-webodev-baris-yas
Vanilla HTML, CSS ve JavaScript ile geliştirilmiş çok sayfalı responsive otel web sitesi. Türkçe/İngilizce dil desteği, görsel lightbox, mobil hamburger menü, scroll reveal animasyonları ve Google Maps entegrasyonu içermektedir. Üniversite ödevi kapsamında geliştirilmiştir.
